# 自定义灯光秀(LightShow)编程文件使用方法
#### 1、准备一个U盘，格式化一下,下面任何一种格式都可以:exFAT, FAT 32 (for Windows), MS-DOS FAT (for Mac), ext3, or ext4
#### 2、下载下面链接的压缩包，解压之后把这LightShow目录拷贝到U盘根目录
#### 3、将U盘插到机车手套箱或者前排U盘接口
#### 4、点击“灯光秀”开始你的表演（显示为自定义灯光秀算是文件识别成功）
